# Placement Tracker 📊

A smart Placement Tracker that helps students organize job applications, track interview progress, and stay fully prepared for campus placements in one place. Built to simplify your placement journey with clear insights, structured tracking, and better decision-making.

## Features
- Track job applications and statuses  
- Manage company and role details  
- Monitor interview progress  
- Dashboard for overall placement overview  
- Secure authentication system  

## Tech Stack
- MongoDB  
- Express.js  
- React.js  
- Node.js  
- JWT Authentication  
- Tailwind CSS  

## Goal
To make placement preparation organized, simple, and effective for students.
